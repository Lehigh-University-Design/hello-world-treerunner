CSS Positioning

Used to alter the element's natural layout flow position on the page and customize it according to our needs. It can be used to fix an element somewhere on the page, give a relative or an absolute position, or no special position at all.

Static

Static positioned elements are not affected by the top, bottom, left, and right properties.

This is a static positioned div

Fixed

This is a fixed div

Relative

Relative is positioned relative to its normal position. It's used best when there's an ancestor with some position property as well and we can alter its final position. Otherwise, it’s just like static.

This div element has position: relative;

Absolute

absolute behaves like fixed except relative to the nearest positioned ancestor instead of relative to the viewport. If an absolutely-positioned element has no positioned ancestors, it uses the document body and still moves along with page scrolling.

Sticky

sticky is a hybrid between relative and fixed position, which allows an element to act like it is relatively positioned until it is scrolled to a certain point.
