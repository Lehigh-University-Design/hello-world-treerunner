Srcset

Srcset defines multiple sizes of the same image, allowing the browser to select the appropriate image source. Srcset is a little bit sketchy sometimes because you are at the whim of the browser. If the browser has a larger image in the cache it will use that image no matter the breakpoint.

Picture

The picture element contains at least one img and can have multiple sources to choose from, based on the display. Picture is handy when you want to change the aspect ratio of an image at a spercific breakpoint.
